WebEx Q&A sessions

Please find below information on WebEx Q&A sessions that the OECD is organising in October. Note that you will need to register for these sessions. To register, please send an email to STIPolicy.data@oecd.org [1] indicating which session you wish to join.

TECHNICAL INFORMATION

Should you be unable to join via Internet (video), please call in using a regular phone and follow the instructions below:

  1. Dial the local call-in number shown on the list of Global call-in numbers [2] that corresponds to the country where you are located, or any other number on the list (if there is no number for your country).
  2. Enter the access code corresponding to your WebEx meeting - see below -  (“xxx xxx xxx”), followed by ##
  3. You will then be directly connected to the meeting space as a Call-In User.

Please also ensure that your computer is equipped with the necessary audio-visual devices (integrated microphone + speakers), or plug into it a headset and webcam. If you are using a smartphone or tablet, note that you will also need to insert the meeting password (“OECD”).
